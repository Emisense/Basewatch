Basewatch by pzion

A Mac OS X time tracker widget for Basecamp

Basewatch is a Mac OS X Dashboard widget for time tracking on 37signals' Basecamp project mamangement system.

Install

Download the latest version of Basewatch from here. Unarchive the downloaded file and then double-click the widget icon to install it in your Dashboard. When Basewatch first starts, it will give the error "No login credentials". You need to provide Basewatch with the URL and API token that you use to access Basecamp. The API token can be found on the "My Info" page of Basecamp. These values are entered on the "flip side" of the widget; click the "i" icon in the bottom-right-hand corner to access the flip side.
